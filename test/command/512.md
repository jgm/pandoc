Multiple indirection:

```
% pandoc -f rst
`click here`__

.. _link1: link2_
.. _link2: http://johnmacfarlane.net/pandoc/

__ link1_
^D

```

Loop detection:

```
% pandoc -f rst
`click here`__

.. _link1: link2_
.. _link2: link1_

__ link1_
^D
[WARNING] Circular reference 'link1' at line 1 column 15
<p><a href="">click here</a></p>
```
