```
% pandoc -t native
[hi]{.smallcaps}
^D
[Para [SmallCaps [Str "hi"]]]
```

```
% pandoc -t native
[hi]{style="font-variant: small-caps;"}
^D
[Para [SmallCaps [Str "hi"]]]
```

```
% pandoc -t native
<span class="smallcaps">hi</span>
^D
[Para [SmallCaps [Str "hi"]]]
```

```
% pandoc -f html -t native

^D
[Para [SmallCaps [Str "hi"]]]
```

```
% pandoc -f html -t native
```
