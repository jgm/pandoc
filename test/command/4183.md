```
% pandoc -f html -t native
<figure>
  <img src="foo" alt="bar">
</figure>
^D
[Para [Image ("",[],[]) [] ("foo","fig:")]]
```

```
% pandoc -f html -t native
<figure>
  <img src="foo" alt="bar">
  <figcaption>
    <div>
      baz
    </div>
  </figcaption>
</figure>
^D
[Para [Image ("",[],[]) [Str "baz"] ("foo","fig:")]]
```

```
% pandoc -f html -t native
<figure>
  <img src="foo">
  <figcaption><p><em>baz</em></p></figcaption>
</figure>
^D
[Para [Image ("",[],[]) [Emph [Str "baz"]] ("foo","fig:")]]
```
