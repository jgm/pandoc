```
% pandoc -f latex -t html
\mbox{abc def} ghi
^D
<p>abc def ghi</p>
```

```
% pandoc -f latex+raw_tex -t native
\mbox{abc def}
^D
[Para [RawInline (Format "latex") "\\mbox{abc def}"]]
```

```
% pandoc -f latex -t html
abc \mbox{\textit{def ghi} jkl} mno
^D
<p>abc <em>def ghi</em> jkl mno</p>
```

```
% pandoc -f latex -t html
abc \mbox{def \\ ghi} jkl
^D
<p>abc defghi jkl</p>
```
