```
% pandoc -f latex -t html
\mbox{abc}
^D
<p><span class="mbox">abc</span></p>
```

```
% pandoc -f latex+raw_tex -t native
\mbox{abc}
^D
[Para [RawInline (Format "latex") "\\mbox{abc}"]]
```

```
% pandoc -f latex -t html
abc \mbox{\textit{def} ghi} jkl
^D
<p>abc <span class="mbox"><em>def</em> ghi</span> jkl</p>
```

```
% pandoc -f latex -t html
abc \mbox{def \\ ghi} jkl
^D
<p>abc <span class="mbox">defghi</span> jkl</p>
```
